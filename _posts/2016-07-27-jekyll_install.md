### jekyll install

+ install ruby
+ install jekyll
    - gem install jekyll
    - gem install jekyll-paginate
    - gem install rdiscount
    - gem install rouge
    - rougify style monokai.sublime > rouge.css
+ start server
    - jekyll build
    - jekyll serve
